El objetivo del reposirorio es documentar y automatizar las tareas de un equipo windows.
Archivo equipo-info.txt, este archivo contiene algunas de las configuraciones de mi equipo
El archivo check-disk.ps1, contiene un script que lo que hace es que te revisa tu disco completamente.
El archivo firewall-rules.conf, contiene un archivos de simulacion con puertos de el firewall.
